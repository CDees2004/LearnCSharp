C# consists of native built in language features like keywords and construct.
.Net is methods and classes requiring a namespace to be stated.

Namespaces are containers to group related code like classes, structs, and enums.
 
C# is typed in PascalCase typically. 

C# supports regions of code using #region Name and #endregion which allows you 
to collapse regions of code in visual studio.

Attaching a suffix to a numeric constant is a quick way of converting the 
default type of the value.
f = float 
d = double 
m = decimal
L = long (not l, looks like a 1) 
u = uint 
uL = unsigned long 

C# is statically typed so everything needs declaration and types before usage 
since types are checked at compile time. 

int number; 
float grade; 
string name; 

while vs do: 

	while(condition){
		// loop body 
	}
	
	do {
		// loop body 
	} while (condition); 
	
While checks condition, then executes. Do executes, then checks condition. 

for: 
	for (init; condition; step){
		// loop body 
	}
	
	for(int i = 0; i < 10; i++){
		Console.WriiteLine(i); 
	}
