Dynamic Arrays:
	Use the List class to create a dynamic array. 
	
	using System.Collections.Generic; 
	
	List<int> numbers = new List<int>(); 
	numbers.Add(5); 
	numbers.Add(7);
	numbers.AddRange(new List<int>{2, -9, 0}); 
	numbers.ForEach(n=>{Console.WriteLine(n);}); 
	
	List<int> numbers2 = new List<int>{1,3,4,5}; 
	
Exception Handling: 
	Used to catch exceptions thrown in code. 
	Can declare your own as long as they inherit from the 
	Exception class 
	
	try {
		// some code that can throw an exception 
	}
	catch (BadCastException e) {
		Console.WriteLine(e.Message);
	}
	catch (IOException e){
	
	}
	
Sampling of Keywords: 
	abstract 
		used to create an abstract class - not implemented 
		directly and can have an incomplete implementation. 
		(method declarations with no body) 
		
		abstract classes are meant to be inherited from with 
		abstract members overwritten. 
		
	as 
		used for safe casting
